# Squad feature

The root session owns the run and creates specialists directly; `agents.max_depth = 1` prohibits delegation chains. Assign non-overlapping write scopes to implementer, SRE, doc-writer, or test-author. Keep reviewers and analysts read-only.

After implementation, invoke `node scripts/workflow-gates.mjs` for lint and test commands. Persist each gate result, request an independent reviewer only after deterministic gates, and update the handoff before finishing.
